# WooCommerce Points & Rewards - Functions.php 點數折抵區塊

這個專案提供了可以直接放置於佈景主題 `functions.php` 檔案中的點數折抵區塊代碼，讓您的購物車和結帳頁面顯示點數使用功能。

## 檔案說明

### `functions-php-code.php` - 完整版
- 完整功能的點數折抵區塊
- 響應式設計，支援手機和平板
- 包含豐富的 CSS 樣式和動畫效果
- 快捷點數使用選項（100點、50%、全部使用）
- 完整的 AJAX 處理和錯誤訊息
- 詳細的點數資訊顯示

### `functions-php-simple.php` - 簡化版
- 基本的點數折抵功能
- 精簡的代碼，易於理解和修改
- 簡單的樣式和界面
- 適合對代碼有基本了解的用戶

## 安裝步驟

1. **確保外掛已安裝**
   - 先安裝並啟用 "WooCommerce Points & Rewards 會員系統" 外掛
   - 確保 WooCommerce 也已正確安裝

2. **外掛設定**
   - 進入外掛設定頁面
   - 啟用「購物車點數折抵」功能
   - 設定最低購物車金額（可選）
   - 設定最大折抵比例（建議 50-100%）

3. **複製代碼**
   - 選擇其中一個版本的代碼
   - 完整複製檔案內容

4. **加入 functions.php**
   - 開啟佈景主題的 `functions.php` 檔案
   - 將代碼貼上到檔案末尾
   - 儲存檔案

## 常見問題

### Q: 點數區塊沒有顯示？
A: 請檢查：
- 外掛是否已啟用
- 外掛設定中是否啟用購物車點數折抵
- 用戶是否已登入
- 購物車金額是否達到最低要求

### Q: 樣式與佈景主題不符？
A: 可以修改 CSS 樣式：
- 完整版：修改 `get_inline_css()` 方法
- 簡化版：修改 `wp_add_inline_style()` 中的 CSS

### Q: 與其他外掛衝突？
A: 這個代碼設計為獨立運作：
- 使用不同的 session 鍵值
- 使用不同的 AJAX action 名稱
- 不會影響原外掛功能

### Q: 如何自訂點數折抵區塊位置？
A: 可以修改 hook 位置：
- 購物車：`woocommerce_cart_totals_before_order_total`
- 結帳：`woocommerce_review_order_after_order_total`
- 更多位置請參考 WooCommerce 官方文件
