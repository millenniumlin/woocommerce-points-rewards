# Contribution Guidelines

Thank you for your interest in contributing to WooCommerce Points & Rewards! We welcome contributions of all kinds, whether it is bug fixes, new features, or documentation improvements. Here are some guidelines to help you get started:

## How to Contribute

1. **Fork the Repository**: Click the "Fork" button at the top right of this page to create a copy of this repository in your account.
2. **Clone your Fork**: Use `git clone` to clone your forked repository.
3. **Create a Branch**: Create a new branch from the `main` branch for your feature or bugfix with `git checkout -b your-branch-name`.
4. **Make Changes**: Make your changes and test them thoroughly.
5. **Commit Your Changes**: Commit your changes with a clear message about what you’ve done.
6. **Push Your Changes**: Push your branch to your forked repository.
7. **Open a Pull Request**: Navigate to the original repository and click on "Compare & pull request" button.

## Guidelines

- **Code Style**: Please follow the coding style used in the project.
- **Testing**: Ensure that your changes are well-tested, and add any necessary tests for new features.
- **Documentation**: If you add new features, please document them properly.

## Issues

If you find any issues or bugs, please report them by creating a new issue.

Thank you for contributing!
