# WooCommerce Points & Rewards

完整的 WooCommerce 累積消費點數獎勵系統外掛，支援會員等級、點數回饋、折抵功能等。

## 功能特色

### 🏆 會員等級系統
- 可自訂會員等級名稱
- 基於年消費累積金額的自動升級機制
- 會員資格有效期限管理（1年）
- 自動降級機制
- 詳細的升降級邏輯（請參考 [會員等級管理說明](docs/MEMBER_TIER_LOGIC.md)）

### 💎 點數回饋系統
- 靈活的點數回饋比例設定
- 會員等級額外回饋百分比
- 註冊贈送點數
- 生日贈送點數 *(關閉再啟用外掛將會重新計算會員生日有無發送)
- 點數有效期限管理

### 🛒 點數折抵功能
- 購物車點數折抵
- 可設定最低購物車金額門檻
- 單筆訂單最大折抵百分比限制
- 靈活的點數兌換率設定

### 📧 通知系統
- 會員資格到期前提醒
- 點數到期前提醒
- 點數獲得通知
- 等級升級通知

### 📊 完整的管理功能
- 儀表板總覽
- 會員等級管理
- 點數記錄查詢
- 詳細報表分析
- 用戶點數管理

## 系統需求

- WordPress 6.0+
- WooCommerce 8.0+
- PHP 8.0+
- MySQL 5.7+ / MariaDB 10.2+

## 安裝說明

1. 下載外掛檔案
2. 上傳到 `/wp-content/plugins/woocommerce-points-rewards/` 目錄
3. 在 WordPress 管理後台啟用外掛
4. 前往 **點數獎勵 > 設定** 進行基本設定

## 快速設定

### 基本點數設定
1. 前往 **點數獎勵 > 設定 > 點數獲得**
2. 設定「每 100 元回饋 1 點」（或您希望的比例）
3. 設定點數有效期限
4. 啟用註冊贈送點數和生日贈送點數

### 會員等級設定
1. 前往 **點數獎勵 > 會員等級**
2. 編輯預設等級或新增自訂等級
3. 設定各等級的消費門檻和額外回饋比例

### 點數使用設定
1. 前往 **點數獎勵 > 設定 > 點數使用**
2. 設定點數兌換比例（通常 1 點 = 1 元）
3. 設定使用門檻和最大折抵比例

## 使用方法

### 前端功能
- 用戶可在 **我的帳戶** 中查看點數餘額和會員等級
- 購物車和結帳頁面可使用點數折抵
- 產品頁面顯示可獲得的點數
