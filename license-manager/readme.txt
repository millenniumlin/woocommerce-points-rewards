授權碼管理系統，用於管理和驗證產品授權。

== Description ==

ML License Manager 是一個功能完整的授權碼管理系統，可以幫助您管理和驗證產品授權。

主要功能：

* 授權碼生成和管理
* 授權碼驗證
* 授權碼有效期限管理
* 授權碼綁定管理
* 完整的管理介面
* REST API 支援

== Installation ==

1. 上傳外掛檔案到 `/wp-content/plugins/license-manager/` 目錄
2. 在 WordPress 管理後台「外掛」選單啟用本外掛
3. 前往「授權管理」選單開始使用

== Frequently Asked Questions ==

= 這個外掛需要什麼系統需求？ =

* WordPress 6.0 或更高版本
* PHP 8.0 或更高版本
* MySQL 5.7 或更高版本

= 如何生成授權碼？ =

啟用外掛後，前往「授權管理」選單，即可在管理介面中生成和管理授權碼。
